# Git'ga yuklash uchun komandalar

Quyidagi komandalarni terminalda ketma-ket bajarib yuboring:

## 1. O'zgarishlarni ko'rish:
```bash
git status
```

## 2. Barcha o'zgarishlarni qo'shish:
```bash
git add .
```

## 3. Commit qilish:
```bash
git commit -m "Add Telegram Mini App with modern design"
```

## 4. GitHub'ga yuklash:
```bash
git push origin main
```

Agar remote repository bo'lmasa:
```bash
git remote add origin https://github.com/yourusername/your-repo.git
git push -u origin main
```
